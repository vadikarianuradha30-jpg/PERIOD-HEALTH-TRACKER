                                                          Period Health Tracker
* Project Overview
Period Health Tracker is a simple application that can help users track their menstrual cycle, predict their next period, and understand their different reproductive health stages.

* Technologies Used
1. Python for implementing the logic
2. HTML, CSS, JavaScript for creating the web application

 * Problem Statement
Most people find it difficult to track their menstrual cycle regularly due to various reasons like lack of awareness, busy schedule, and lack of access to simple tracking tools.

* Project Objectives
1. To develop a simple period tracking application
2. To predict the dates for the next period
3. To calculate the fertile window
4. To increase awareness about women’s health
5. To develop a simple application using simple technologies for beginners

* Solution Overview
The application can help users track their menstrual cycle by simply entering their last period dates and their average cycle length to know their next period dates, fertile window, and current cycle phase.

 Application Features
* Core Features
1.  Next period prediction
2. Menstrual phase detection
3.  Fertile window calculation
4. Fertile days identification

* How to Run the Project

 Web Version
1. Open the folder containing the project
2. Double click on `index.html`
3. Input:
   - Last period date
   - Cycle length
4. Click on `Predict`
5. Results displayed instantly

Python Version
1. Open the project in VS Code
2. Open Terminal
3. Run
4. Input of values at respective prompts
5. Viewing output on terminal screen

 Working of the System

1. Input from the user:
   - Last period date
   - Cycle length

2. Process performed on the input:
   - Next period date calculation
   - Fertile window calculation (approx. 14 days prior to next cycle start)
   - Current phase calculation (approx.)

3. Output displayed to the user:
   - On screen (for web version)
   - On terminal screen (for Python version)
